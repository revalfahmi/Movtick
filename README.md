# Movtick
Movtick is an application for booking movie tickets.

## Feature
1. Splash Screen
2. Onboarding
3. Sign In
4. Sign Up
5. Choose Film
6. Choose Seat
7. Edit Profile
8. Take a picture for the profile photo during registration
9. History 
10. Scan QR Code or Barcode
